English--
This repository contains the SinusBot Script Library OKlib and the JSDoc based javascript file from which the documentation can be generated.
Sinusbot and oklib are required for this script to work.
Provides automatic definition of other authorizations when an authorization is given on Teamspeak

Türkçe--
Bu depo, SinusBot Komut Dosyası Kitaplığı OKlib'i ve belgelerin oluşturulabileceği JSDoc tabanlı javascript dosyasını içerir.
Scriptin çalışması için oklib kütüphanesi ve sinusbot ekosistemi gereklidir.
Teamspeak üzerinde bir yetki verildiğinde diğer seçilmiş yetkilerin otomatik verilmesini sağlar.
